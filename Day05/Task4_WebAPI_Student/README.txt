Student Web API (.NET 6+)
------------------------------------------
Endpoints:
GET /api/students -> Display all students
GET /api/students/{id} -> Display student by ID
GET /api/students/range?minMarks=50&maxMarks=90 -> Display students in range

Instructions:
1. Create new ASP.NET Core Web API project.
2. Add 'Student' model and 'StudentController'.
3. Use Entity Framework Core for database operations.
