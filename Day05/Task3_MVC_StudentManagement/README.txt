Student Management System (ASP.NET MVC + Entity Framework)
-----------------------------------------------------------
Models/Student.cs:
- Id, Name, BirthDate, Address, SchoolName, Percentage

Controllers/StudentController.cs:
- CRUD Operations (Index, Details, Create, Edit, Delete)

Views/Student:
- Razor views for all CRUD actions

Setup Steps:
1. Create ASP.NET MVC project in Visual Studio.
2. Add Entity Framework (Code First).
3. Replace model/controller/view code with the provided structure.
4. Update connection string in Web.config.
